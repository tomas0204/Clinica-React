
# Bienestar360 - Clinica Medica
Bienvenidos a nuestro proyecto del segundo modulo de programacion 
Nuestro tema elegido para este proyecto fue Clinica Medica ⚕️

## 📌 Descripción del proyecto
Esta es una aplicación web para una clínica médica, desarrollada con React, que centraliza distintas funciones tanto para pacientes como para profesionales de la salud.
El sistema busca mejorar la organización interna y ofrecer herramientas accesibles desde cualquier dispositivo.

La plataforma cuenta con tres módulos principales:

### 🔐 1. Login para pacientes y médicos

El sitio dispone de un sistema de inicio de sesión donde cada usuario accede a funcionalidades según su rol:
Pacientes: pueden ingresar para visualizar su información personal y acceder a su historia clínica.
Médicos: pueden acceder al panel profesional y visualizar sus propias secciones habilitadas.

### 📄 2. Visualización de Historia Clínica

Cada paciente puede consultar su historia clínica, que incluye información médica relevante.
La interfaz es clara, ordenada y accesible, permitiendo que el paciente vea su información sin necesidad de acudir presencialmente.

### ⚕️ 3. Gestión de Guardia Médica

El módulo de guardia médica permite:

**Administradores:** crear, editar, visualizar y eliminar médicos de guardia. <br>
**Usuarios visitantes:** ver qué profesionales están atendiendo en turnos actuales.

## 👨‍⚕️🧑‍💻🙍Roles
<strong>Administrador:</strong> Puede crear, modificar y eliminar. <br>

<strong>Profesional de la Salud:</strong> Puede crear, modificar y eliminar tambien, pero solo lo que tenga que ver con su labor.   

<strong>Visitante:</strong> Solo puede visualizar.
